# AIなぞかけメーカー

## 概要
ユーザーが入力したお題に対して、生成AIが2つのなぞかけを提示。評価を送信し、履歴を管理できます。

## 公開URL
- フロントエンド（PWA）：https://nazokake-maker.vercel.app
- バックエンドAPI：https://nazokake-api.onrender.com

## セットアップ方法
1. Node.jsをインストール
2. `npm install` で依存関係を導入
3. `npm run dev` または `npm run start` で起動
